This plugin provided users to easily add a custom user registration & login  form anywhere on their site using simple shortcodes.

== Description ==
This plugin creates a user interface for adding custom registration forms to any post or page on your WordPress website. After Installation, you will find a new menu called 'LRMP Options', You can copy the shortcodes from there also. Simply navigate to the edit page for post .
Admin Custom Login & Registration Form plugin give ability to customize your WordPress login & registration Form page according to you. 

> **provided Shortcode**
>
> * login form Shortcode: [lrmp_login_form] 
> * Registration Form Shortcode: [lrmp_registration_form]
>

> **Standard Features**
>
> * Add Your Custom Login Form any post/pages.
> * Login Form Colour
> * Login Form Font Colour
> * Login Form Font Size
> * Login Form Button Colour
> * Login Form Button Size
> * Login Form with Font Awesome Icons
>

== Installation ==

1. Download ` Login and Registration Master Pro` and upload contents to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Insert forms using the Shordcode registration button on your post/page.



### Responsive Design:
Plugin Design is totally mobile compatible.
